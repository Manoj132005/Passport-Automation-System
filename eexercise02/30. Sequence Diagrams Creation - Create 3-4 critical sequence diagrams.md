# Sequence Diagrams  
## Passport Automation System

---

## 1. User Registration & Login Sequence Diagram

### Description
This sequence diagram shows how a user registers and logs into the system using valid credentials.

### Sequence Steps
1. User enters registration details
2. System validates and stores user data
3. User logs in with credentials
4. System authenticates and grants access

---

## 2. Passport Application Submission Sequence Diagram

### Description
This diagram illustrates the process of submitting a passport application.

### Sequence Steps
1. User fills the passport application form
2. System stores application details
3. Application ID is generated
4. User receives confirmation

---

## 3. Document Upload & Verification Sequence Diagram

### Description
This diagram explains how documents are uploaded by the user and verified by the passport officer.

### Sequence Steps
1. User uploads required documents
2. System stores documents
3. Officer verifies documents
4. Verification status is updated

---

## 4. Police Verification & Passport Approval Sequence Diagram

### Description
This sequence diagram represents police verification and final passport approval.

### Sequence Steps
1. Officer requests police verification
2. Police department submits verification report
3. Officer reviews application
4. Passport approval notification is sent to the user
