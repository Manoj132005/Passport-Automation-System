# Problem 34: Workflow Documentation

## Overview
The workflow of the Passport Automation System was documented to represent the step-by-step process followed during passport application, verification, and approval.

## Workflow Steps
1. Applicant registers and logs into the system  
2. Applicant fills passport application form  
3. Applicant uploads required documents  
4. Applicant books an appointment  
5. Passport Officer verifies the application  
6. Application is approved or rejected  
7. Notification is sent to the applicant  

## Purpose
Workflow documentation helps in understanding the operational flow of the system and ensures clarity in process execution among all stakeholders.
