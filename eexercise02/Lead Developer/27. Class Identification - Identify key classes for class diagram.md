# Passport Automation System - Class Identification

## 1. User / Applicant Classes

### Applicant
- **Attributes:**
  - `applicantID`
  - `name`
  - `dob`
  - `gender`
  - `address`
  - `email`
  - `phoneNumber`
  - `passportType`
  - `applicationStatus`
- **Methods:**
  - `applyForPassport()`
  - `updateProfile()`
  - `checkApplicationStatus()`
  - `uploadDocuments()`

### Login / UserAuthentication
- **Attributes:**
  - `username`
  - `password`
  - `role`
- **Methods:**
  - `login()`
  - `logout()`
  - `changePassword()`
  - `validateUser()`
