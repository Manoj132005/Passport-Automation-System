# Problem 21: Team Workspace Setup

## Overview
For the SEAD project titled **“Passport Automation System”**, a collaborative team workspace was established to support coordinated development, documentation, and design activities among team members.

## Workspace Tools Used
- **GitHub** – Version control and document management  
- **StarUML** – UML diagram creation and modeling  
- **Google Drive** – Sharing reference materials  
- **Communication Tools** – WhatsApp and Google Meet  

## Team Roles in Workspace
- **Manoj V** – Project Manager  
- **Dharshini M** – Lead Developer  
- **Bhuvana S** – Documentation Lead  
- **Suriya SM** – QA Lead  

## Purpose
The workspace was configured to:
- Enable parallel work on system design and documentation  
- Maintain version control for UML models and reports  
- Ensure consistent communication among team members  
- Track updates and changes efficiently
