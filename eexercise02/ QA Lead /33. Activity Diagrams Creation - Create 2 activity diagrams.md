# Problem 33: Activity Diagram

## Overview
An activity diagram was created for the Passport Automation System to represent the flow of activities involved in the passport application process from start to end.

## Activities
- Register / Login  
- Fill Application Form  
- Upload Documents  
- Book Appointment  
- Verify Application  
- Approve or Reject Application  
- Send Notification  

## Purpose
The activity diagram helps visualize the sequential and decision-based flow of actions performed by the system and users during passport processing.
