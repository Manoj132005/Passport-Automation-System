## Alternative Flows Inclusion  

---

## 1️. Alternative Flow: User Registration & Login Failure

### Primary Flow
- User registers or logs in successfully.
- System authenticates credentials.
- User dashboard is displayed.

### Alternative Flows
- **Invalid credentials**
  - Authentication Service rejects login.
  - Error message is shown to the user.
- **User already exists**
  - Registration request is denied.
  - System prompts user to log in instead.
- **Server or database error**
  - System displays temporary failure message.

### Impact
Ensures secure access and prevents unauthorized login.

---

## 2️. Alternative Flow: Passport Application Submission

### Primary Flow
- User fills application form.
- Documents are uploaded.
- Application is submitted successfully.

### Alternative Flows
- **Incomplete application**
  - System detects missing fields.
  - User is prompted to complete details.
- **Invalid document format**
  - Upload is rejected.
  - User is asked to re-upload documents.
- **Payment failure**
  - Application submission is paused.
  - User retries payment.

### Impact
Prevents invalid or incomplete applications from entering the system.

---

## 3️. Alternative Flow: Document Verification

### Primary Flow
- Documents are uploaded.
- Passport Officer verifies documents.
- Verification status is updated.

### Alternative Flows
- **Document mismatch**
  - Verification fails.
  - User is asked to upload corrected documents.
- **Poor document quality**
  - Officer rejects documents.
  - Re-upload request is generated.
- **Verification delay**
  - Status marked as “Pending”.

### Impact
Ensures authenticity and correctness of submitted documents.

---

## 4️. Alternative Flow: Police Verification & Passport Approval

### Primary Flow
- Police verification is successful.
- Passport is approved.
- User is notified.

### Alternative Flows
- **Negative police report**
  - Application is rejected.
  - Rejection reason is stored.
- **Verification not completed**
  - Application status set to “On Hold”.
- **_ATTACHMENT REQUEST**
  - Additional documents requested from user.

### Impact
Maintains national security and regulatory compliance.
