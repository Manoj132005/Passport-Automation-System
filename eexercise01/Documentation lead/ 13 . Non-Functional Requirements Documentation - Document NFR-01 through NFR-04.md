# Non-Functional Requirements

## NFR-01: Performance
The system should respond to user actions within 3 seconds under normal load.

## NFR-02: Security
The system must encrypt sensitive user data and use secure authentication.

## NFR-03: Availability
The system should be available 24/7 with at least 99% uptime.

## NFR-04: Usability
The system should be easy to use with a simple and clear interface.

## NFR-05: Scalability
The system should support an increasing number of users without performance issues.

## NFR-06: Maintainability
The system should be easy to update and maintain by technical staff.
