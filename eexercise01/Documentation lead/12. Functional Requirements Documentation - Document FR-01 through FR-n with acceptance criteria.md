# Functional Requirements

## FR-01: User Registration
The system shall allow new users to create an account using basic personal details.

## FR-02: User Login
The system shall allow registered users to log in securely.

## FR-03: Passport Application Submission
The system shall allow users to fill and submit a passport application form online.

## FR-04: Document Upload
The system shall allow users to upload required documents (ID proof, address proof, etc.).

## FR-05: Appointment Scheduling
The system shall allow users to book an appointment for verification.

## FR-06: Application Status Tracking
The system shall allow users to view the current status of their application.

## FR-07: Admin Verification
The system shall allow passport officers/admins to verify submitted applications.

## FR-08: Approval or Rejection
The system shall allow admins to approve or reject applications with remarks.

## FR-09: Notification System
The system shall notify users about status updates via email or SMS.
