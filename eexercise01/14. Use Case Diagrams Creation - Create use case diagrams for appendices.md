## Actors
- Applicant  
- Passport Officer  
- System Administrator  

## Main Use Cases
- Register Account  
- Login  
- Apply for Passport  
- Upload Documents  
- Book Appointment  
- Track Application Status  
- Verify Application  
- Approve/Reject Application  
- Send Notifications  

## Description
The use case diagram represents the interaction between users (Applicants and Admins) and the Passport Automation System.  
It shows how applicants submit requests and how officers verify and approve them.
